### ASCII Ray-Tracer

Ray-tracing a scene and "printing" the result to the console.
Terminal settings:
- Font: Liberation Mono (for ok dimensions, spheres are still a bit ellipsoid^^)
- Size: 1 (for high res mode)

#### Loop
`./raytracer -width=160 -height=90 -output=loop`
